APIs

Here are a list of APIs that we call the make our development process easier!

Name: createInput: function (id, title, type, location)

Description: This API is going to help developers to create Input, the params that developers should give is that the id of the input, tag of the input, type of the input such as "password" or "text" and the location the input module you want to attach to.

Name: createSubmit: function (type, value, location)

Description: This API is going to help developers to create Submit Button, the params that developers should give is that the type of the submit(we can just use "submit"), value of the submit,and the location the submit module you want to attach to

Name: createButton: function (value, location)

Description: This API is going to help developers to create Button, the params that developers should give is that the value of the submit,and the location the submit module you want to attach to

Name: createAvatar: function (src)

Description: This API is going to help developers to create Avatar in our module, the params that developers should give is that src of the avatar.

Name: loginModule: function ()

Description: This API is a predefined loginModule.

Name: registerModule: function ()

Description: This API is a predefined registerModule.

Name: addSubmitAnimation: function (result)

Description: This API is going to add Login and Register Animation based on user's input, the param that developers should give the the result("success" or "fail") of the login and register.

Name: easyLogin: function (top, left, avatar_src, background_src)

Description: This API is going to create predefined EasyLogin Module. The params developer should define is the percentage to the top, the percentage to the left and the src of avatar and background picture src.

Name: changeBackground(src)

Description: This API is going to change the background picture of the screen. We should define the src.

function inputListener(id, type)

Description: This API is going to validate users' inputs. The params we need to give is the "id" of the textbox we want to validate, and the type of the input such as (password, text, email, etc). This API will call its sub functions to validates input and give hints.
