Example

As the name suggests, how to apply this Login and Register library aims to be as easy as possbile. Developers simpiliy just need to import the "easyLogin.js", "inputValidator.js", and ”index.css“. And then, after this import process, developers need to create a seperate js file to create the Ojbect and call the API

Here is a js example about how to create and customize your "EasyLogin" module

const lg = new LoginGenerator();
lg.easyLogin("35%", "10%", "logo.png", "background.jpg");
lg.addSubmitAnimation("success");

lg.easyLogin("35%", "10%", "logo.png", "background.jpg");
is going to create a Pop-up EasyLogin window with some animations with "35%"" from the top , 10% form the left. "logo.png" is the the Avatar src and "background.jpg" is the background picture src. The user can customize their module based on their needs.

lg.addSubmitAnimation("success");
is going to add login and register button animation, developers can integrate this api call with their code. For example, if the login is successful, they can call lg.addSubmitAnimation("success"). Or, if the login is unsuccessful, they can call lg.addSubmitAnimation("fail") to add the failure animation.

The example above is about how to create a predefined EasyLogin Module. It is very simply and easy to use. The developers can also choose to create their own template.

Developers can take a look at the code and API documentation of "loginModule", "registerModule" and "easyLogin". These parts are the format for us to create own template and very easy to understand. This library aims to give enough flexablity for developers to create their own template by giving them enough precreated helper APIs. Developers can choose to add customized numbers of input box, and various kinds of input validation schema to the input box
